# Research Reports

Generate interactive dashboards to analyze strategy behavior, detect patterns, and validate hypotheses.

---

## Report Categories

### Card Reports

Display aggregated metrics as dashboard cards.

#### Numeric Cards

<!-- snippet: type=complete_code, needs_data=single_asset, timeframe=1D, complexity=medium -->
```python
src = market_data_source(timeframe="1D")
returns = roc(period=1)(src.c)

# Single metric card
numeric_cards_report(agg="mean", category="Performance", title="Avg Daily Return")(returns)

# Multiple metrics in one category
numeric_cards_report(agg="mean", category="Stats", title="Mean", group=0, group_size=3)(returns)
numeric_cards_report(agg="stddev", category="Stats", title="Volatility", group=0, group_size=3)(returns)
numeric_cards_report(agg="max", category="Stats", title="Max Return", group=0, group_size=3)(returns)
```

**Aggregations:** `mean`, `sum`, `min`, `max`, `stddev`, `variance`, `first`, `last`, `count`, `median`, `skew`, `kurtosis`

#### Boolean Cards

<!-- snippet: type=complete_code, needs_data=single_asset, complexity=simple -->
```python
src = market_data_source(timeframe="1D")
rsi_val = rsi(period=14)(src.c)
signal = crossover(rsi_val, 30)
boolean_card_report(category="Signals", title="RSI Crossover Count")(signal)
```

#### Index Cards

Show specific rows by index (first, last, percentiles).

<!-- snippet: type=complete_code, needs_data=single_asset, complexity=simple -->
```python
src = market_data_source(timeframe="1D")
returns = roc(period=1)(src.c)
# Filter high return days
high_return_signal = returns > 0.02
index_card_report(index="first", category="Top Days", title="First Occurrence")(returns)
```

#### Quantile Cards

Display quantile values (25th, 50th, 75th percentiles).

<!-- snippet: type=complete_code, needs_data=single_asset, complexity=simple -->
```python
src = market_data_source(timeframe="1D")
returns = roc(period=1)(src.c)
index_card_report(index="p95", category="Risk", title="95th Percentile Return")(returns)
```

### Chart Reports

#### Bar Charts

Categorical comparisons and distributions.

<!-- snippet: type=complete_code, needs_data=single_asset, complexity=simple -->
```python
src = market_data_source(timeframe="1D")
rsi_val = rsi(period=14)(src.c)
returns = roc(period=1)(src.c)

# Distribution by regime
regime = conditional_select(rsi_val < 30, "Oversold", rsi_val > 70, "Overbought", "Neutral")
bar_chart_report(agg="count", title="Bar Count by Regime", category="Distribution")(regime)

# Aggregate metrics by category
bar_chart_report(agg="mean", title="Avg Return by Regime", category="Performance")(regime, returns)
```

**Aggregations:** `count`, `sum`, `mean`, `first`, `last`, `min`, `max`

#### Pie Charts

Proportional breakdowns.

<!-- snippet: type=complete_code, needs_data=single_asset, complexity=simple -->
```python
# Simple proportion
pie_chart_report(agg="count", title="Regime Distribution", category="Analysis")(regime)

# Nested pie (two-level hierarchy)
# Sector categorization
sector = conditional_select(rsi_val < 20, "A", rsi_val < 40, "B", rsi_val < 60, "C", "D")
nested_pie_chart_report(agg="count", title="Asset Distribution", category="Breakdown")(sector, regime)
```

#### Histograms

Value distributions and frequency analysis.

<!-- snippet: type=complete_code, needs_data=single_asset, complexity=simple -->
```python
src = market_data_source(timeframe="1D")
returns = roc(period=1)(src.c)
histogram_chart_report(bins=30, title="Return Distribution", category="Stats", vertical=False)(returns)
```

### Table Reports

Filtered row-level data with custom columns.

<!-- snippet: type=complete_code, needs_data=single_asset, timeframe=1D, complexity=medium -->
```python
src = market_data_source(timeframe="1D")
rsi_val = rsi(period=14)(src.c)
returns = roc(period=1)(src.c)

# Filter extreme conditions
extreme = (rsi_val < 30) | (rsi_val > 70)
regime = conditional_select(rsi_val < 30, "Oversold", "Overbought")

# Boolean filter (extreme)
table_report(schema=TableReportSchema(
    title="RSI Extremes",
    select_key=SLOT0,
    columns=[
        TableColumnSchema(column_id=SLOT1, title="Regime"),
        TableColumnSchema(column_id=SLOT2, title="RSI"),
        TableColumnSchema(column_id=SLOT3, title="Return %")
    ]
))(extreme, regime, rsi_val, returns)
```

**Key Points:**
- `select_key`: Boolean column for row filtering
- Inputs map to `SLOT0`, `SLOT1`, `SLOT2`, etc. by position
- Only rows where `select_key` is `true` appear in table

### Specialized Reports

#### Gap Report

Comprehensive gap analysis with cards, histograms, and pivot tables.

<!-- snippet: type=complete_code, needs_data=single_asset, timeframe=1Min, needs_sessions=true, needs_intraday=true, complexity=medium -->
```python
gaps = session_gap(fill_percent=100, timeframe="1Min")()

# gap_filled: Boolean, gap_retrace: Boolean, gap_size: Decimal, psc: Decimal, psc_timestamp: Timestamp
gap_report(fill_time_pivot_hour=12, histogram_bins=20)(
    gaps.gap_filled,
    gaps.gap_retrace,
    gaps.gap_size,
    gaps.psc,
    gaps.psc_timestamp
)
```

**Output:**
- Summary cards: fill rate, avg gap size, median fill time, total gaps
- Histograms: gap size distribution, fill time distribution
- Pivot tables: gaps by hour, by day of week, by size bucket
- Detail table: timestamp, size, fill time, filled status

## Cross-Sectional Reports

Multi-asset comparison reports for factor strategies.

### CS Bar Chart

Compare all assets in one chart.

<!-- snippet: type=complete_code, needs_data=multi_asset, timeframe=1D, complexity=complex -->
```python
# Sector performance comparison
src = market_data_source(timeframe="1D")  # 11 sector ETFs
returns = roc(period=20)(src.c)

cs_bar_chart_report(agg="last", title="20-Day Returns by Sector", category="Performance")(returns)
# One bar per sector: XLK, XLF, XLE, etc.
```

**Use Cases:**
- Sector performance rankings
- Factor score comparisons
- Multi-asset metrics side-by-side

### CS Table

Asset-by-metric table with aggregation.

<!-- snippet: type=complete_code, needs_data=multi_asset, complexity=complex -->
```python
momentum = roc(period=60)(src.c)
volatility = atr(period=14)(src.c)

cs_table_report(agg="last", title="Sector Metrics", category="Analysis")(momentum, volatility)
# Table: Asset | Momentum | Volatility
#        XLK  |   +15%   |    0.8
#        XLF  |    +8%   |    0.6
```

**Aggregations:** `first`, `last`, `mean`, `sum`, `min`, `max`

### CS Numeric Cards

Card group with one card per asset.

<!-- snippet: type=complete_code, needs_data=multi_asset, timeframe=1D, complexity=complex -->
```python
src = market_data_source(timeframe="1D")
spx = common_indices(index="SPX")

# Calculate returns
returns = roc(period=1)(src.c)
spx_returns = roc(period=1)(spx.c)

# Calculate beta for each asset
sector_beta = beta(window=60)(returns, spx_returns)

cs_numeric_cards_report(agg="last", category="Beta Analysis", group=0, group_size=11)(sector_beta)
# Creates 11 cards showing current beta for each sector
```

**Use Cases:**
- Multi-asset metric dashboards
- Factor exposure visualization
- Comparative performance cards

## Research Workflows

### Pattern Discovery

<!-- snippet: type=complete_code, needs_data=single_asset, timeframe=1Min, needs_sessions=true, needs_intraday=true, complexity=medium -->
```python
# Identify gap fill characteristics
gaps = session_gap(fill_percent=100, timeframe="1Min")()

# Distribution analysis
histogram_chart_report(bins=20, title="Gap Size Distribution", category="Research")(gaps.gap_size)

# Detailed inspection
gap_report(fill_time_pivot_hour=12)(
    gaps.gap_filled, gaps.gap_retrace, gaps.gap_size, gaps.psc, gaps.psc_timestamp
)

# Summary card
numeric_cards_report(agg="count_all", title="Total Gaps", category="Summary", group=0, group_size=1)(gaps.gap_filled)
```

### Factor Analysis

<!-- snippet: type=complete_code, needs_data=multi_asset, timeframe=1D, complexity=complex -->
```python
src = market_data_source(timeframe="1D", symbols=["XLK", "XLF", "XLE", "XLV", "XLI", "XLP", "XLY", "XLB", "XLU", "XLRE", "XLC"])
spx = common_indices(index="SPX")

# Calculate factors
momentum = cs_zscore()(roc(period=60)(src.c))
volatility = cs_zscore()(atr(period=14)(src.c))

# Compare factors across assets
cs_table_report(agg="last", title="Factor Scores", category="Cross-Sectional")(momentum, volatility)

# Visualize distributions
histogram_chart_report(bins=20, title="Momentum Distribution", category="Research")(momentum)

# Summary card
cs_numeric_cards_report(agg="last", category="Summary", group=0, group_size=11, title="Latest Momentum")(momentum)
```

### Regime Detection

<!-- snippet: type=complete_code, needs_data=single_asset, timeframe=1D, complexity=medium -->
```python
src = market_data_source(timeframe="1D", symbols=["AAPL"])
returns = roc(period=1)(src.c)

rsi_val = rsi(period=14)(src.c)
regime = conditional_select(rsi_val < 30, "Oversold", rsi_val > 70, "Overbought", "Neutral")

# Proportion analysis
pie_chart_report(agg="count", title="Market Regime Distribution", category="Research")(regime)

# Performance by regime
bar_chart_report(agg="mean", title="Avg Return by Regime", category="Research")(regime, returns)

# Detailed regime table
extreme = (rsi_val < 30) | (rsi_val > 70)
table_report(schema=TableReportSchema(
    title="Extreme Conditions",
    select_key=SLOT0,
    columns=[
        TableColumnSchema(column_id=SLOT1, title="Regime"),
        TableColumnSchema(column_id=SLOT2, title="RSI")
    ]
))(extreme, regime, rsi_val)

# Summary card
numeric_cards_report(agg="mean", title="Avg RSI", category="Summary", group=0, group_size=1)(rsi_val)
```
