# Visualization System

## Overview

EpochScript provides **automatic visualization** for strategy development. Write your strategy code and visualization happens automatically—no configuration required.

**Three Core Areas:**

1. **Auto-Plotting**: Every indicator automatically appears on charts
2. **Event Markers**: Timeline navigation for signals and patterns
3. **Research Reports**: Interactive dashboards for gap analysis, tables, and charts

**Zero Configuration:**
- Write strategy → Visualization appears
- Add indicator → Chart updates automatically
- Change parameters → See changes instantly

## Quick Start

**Basic strategy with visualization:**

```python
src = market_data_source()

# These automatically plot on chart
ema_20 = ema(period=20)(src.c)        # → Line overlay
rsi_val = rsi(period=14)(src.c)       # → Panel below chart

# Mark RSI crossover events with colored badges
cross_above_30 = crossover(rsi_val, 30)
cross_below_70 = crossunder(rsi_val, 70)
flag = cross_above_30 or cross_below_70
label = conditional_select(
    cross_above_30, "RSI Cross Above 30 (Bullish)",
    cross_below_70, "RSI Cross Below 70 (Bearish)")

event_marker(
    event_marker_schema=EventMarkerSchema(
        title="RSI Crossovers",
        icon=TrendUp,
        select_key=SLOT0,
        schemas=[
            CardColumnSchema(
                column_id=SLOT1,
                slot=Hero,
                render_type=Badge,
                color_map={Success: ["RSI Cross Above 30 (Bullish)"], Error: ["RSI Cross Below 70 (Bearish)"]}
            )
        ]
    )
)(flag, label)
```

**Result:**
- EMA line overlays on price chart
- RSI plots in separate panel (0-100 scale)
- RSI crossover events appear in timeline with color-coded badges
