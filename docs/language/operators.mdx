# Operators

EpochScript supports standard operators for arithmetic, comparison, and logic, plus a special lag operator `[N]` for historical data access.

---

## All Operators

| Category | Operator | Description | Example |
|----------|----------|-------------|---------|
| **Arithmetic** | `+` | Addition | `a + b` |
| | `-` | Subtraction | `a - b` |
| | `*` | Multiplication | `a * b` |
| | `/` | Division | `a / b` |
| | `%` | Modulo | `a % b` |
| | `**` | Power | `a ** b` |
| | `-` (unary) | Negation | `-a` |
| **Comparison** | `>` | Greater than | `a > b` |
| | `<` | Less than | `a < b` |
| | `>=` | Greater or equal | `a >= b` |
| | `<=` | Less or equal | `a <= b` |
| | `==` | Equal | `a == b` |
| | `!=` | Not equal | `a != b` |
| **Logical** | `and` or `&` | Logical AND | `a and b` |
| | `or` or `\|` | Logical OR | `a or b` |
| | `not` or `!` | Logical NOT | `not a` |
| **Ternary** | `if...else` | Conditional | `x if cond else y` |
| **Lag** | `[N]` | N bars ago (N > 0) | `price[5]` |
| **Access** | `.` | Attribute access | `src.c` |

---

## Operator Precedence

Follows [Python's operator precedence](https://docs.python.org/3/reference/expressions.html#operator-precedence) exactly.
